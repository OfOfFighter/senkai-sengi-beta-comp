## 千怪戦戯 デジタルカードシステム ベータ版

### 概要
ダイソーのオリジナルカードゲーム「千怪戦戯」を、Google AI Studioと協力してデジタルカードゲーム化しました。

---

### メインメニューに戻る方法
メインメニューに戻りたい場合は、ページをリロード（再読み込み）してください。

### 画面レイアウトの変更
スタート画面の右上から画面レイアウトを変更できます。

*   **SP**：スマートフォン向けの縦画面に最適化されたモードです。
*   **PC**：PC向けの横画面に最適化されたモードです。

---

### メインメニューの説明

| 機能 | 説明 |
|:---:|:---|
| **CPUと対戦** | CPUとの対戦ができます（現在は弱い設定です）。バグの発見にご協力ください。先攻・後攻はランダムに決定されます。 |
| **対人戦** | 1人での動作確認や練習ができます。メニューから先攻・後攻を選択し、2人分のカードを操作してください。 |
| **デッキ構築** | オリジナルのデッキを作成できます。 |
| **インポート** | 「デッキ構築」で作成し、エクスポートしたJSONファイルを読み込むことができます。読み込んだデッキは、対戦時のデッキ選択画面に追加されます。 |

---

### 注意事項
*   現在はスターターデッキのカードのみ利用可能です。動作の安定性を確認次第、順次カードを追加する予定です。
*   カードの画像は現在非表示にしています（実装検討中）。
*   作成したデッキは、「エクスポート」ボタンでJSONファイルとして保存し、「インポート」機能で読み込んで使用してください。

デッキ構築画面はPC版が見やすくてお勧めです。エクスポートボタンでJSONファイルとして書き出し、インポートで読み込んでデッキを使うことができます
